# BantuPay Sdk Java

In your pom.xml file:

```xml
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
</repositories>

<dependency>
	    <groupId>com.github.bantublockchain</groupId>
	    <artifactId>bantupay-sdk-java</artifactId>
	    <version>1.0</version>
</dependency>
```
